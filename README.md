Smart Health Monitoring System

Overview
This project is a simple Smart Health Monitoring System that simulates real-time temperature and heart rate data. It includes an alert system, live location tracking, and graphical visualization.

Features
- Real-time temperature monitoring
- Heart rate tracking
- Alert system (Normal, Warning, Critical)
- Live location tracking using geolocation
- Data visualization using Chart.js
- Automatic updates every 2 seconds

Technologies Used
- HTML

How to Run
1. Download or clone the repository
2. Open the "index.html" file in any web browser
3. The system will start automatically

Future Improvements
- Integration with real sensors (ESP32 or Arduino)
- Cloud database integration (Firebase or MongoDB)
- Mobile application support
- User authentication and data security
- AI-based health prediction
